2020/04/20

I’ve decided to create a post where I can regularly update some favorite data visualizations. Where I’ve previously discussed the data or shared code I will provide a link. Often I’ll update the charts and post them on Twitter soon after the data is released. I won’t be updating these that quickly, but I’ll do my best to keep up.

As I update some more charts I may add to the list. I might not update with every release, so I’ll indicate the last update for each chart.

Your favorite chart isn’t here? Let me know and maybe I can add it.
